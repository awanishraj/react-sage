# react-sage

Handy-dandy hooks.

## Use

```bash
# Install
npm i react-sage
```

```jsx
// Import specific hooks one by one.
import { useForm } from 'react-sage/useForm'
import { useQuery } from 'react-sage/useQuery'
import { useMutation } from 'react-sage/useMutation'
```

## Demo

```bash
# Starts dev server on http://localhost:1234
npm run demo
```
